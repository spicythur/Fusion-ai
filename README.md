# Fusion AI

AI-powered text-to-3D model generator for Autodesk Fusion 360. Describe what you need in plain language — gears, brackets, drone frames — and get a ready-to-use 3D model.

## Tech Stack

- TypeScript
- Express
- React

## Features

- Natural language to 3D model generation
- Autodesk Fusion 360 integration
- Supports mechanical parts: gears, brackets, frames

## Getting Started

```bash
npm install
npm run dev
```
